## At a glance

- Single-line text field that forwards every native TextInput prop (`value`, `onChangeText`, `placeholder`, `keyboardType`, `secureTextEntry`, …) untouched.
- Pair with [`Field`](/docs/components/field) for label, description, error wiring, and accessible `id` / `aria-*` attributes.
- Compose with [`InputGroup`](/docs/components/input-group) when you need fixed prefix / suffix decorators (@username, https://, USD).

## Use with Field

For labelled fields with description, error, and a11y wiring, wrap in `<Field>`:

```tsx
import { Field, TextInput } from '@nori-ui/core';

export const Example = () => (
    <Field>
        <Field.Label>Email</Field.Label>
        <Field.Description>We won't share it.</Field.Description>
        <Field.Control>
            <TextInput placeholder="you@example.com" />
        </Field.Control>
    </Field>
);
```

With validation error:

```tsx
import { Field, TextInput } from '@nori-ui/core';

export const Example = () => (
    <Field error="That doesn't look like a valid email address.">
        <Field.Label>Email</Field.Label>
        <Field.Control>
            <TextInput placeholder="you@example.com" />
        </Field.Control>
        <Field.Error />
    </Field>
);
```

See [Field](/docs/components/field) for the full API.

## `disabled`

Greys the field, blocks input, and forwards `aria-disabled`. The label
keeps its color so the disabled control still reads as a labeled
field.

```tsx
import { Field, TextInput } from '@nori-ui/core';

export const Example = () => (
    <Field disabled>
        <Field.Label>Email</Field.Label>
        <Field.Control>
            <TextInput defaultValue="locked@example.com" />
        </Field.Control>
    </Field>
);
```

## `leading` and `trailing`

Inline visual decorations *inside* the field — typically icons or short
labels. Unlike [`InputGroup`](/docs/components/input-group) addons,
these don't get their own border; they sit flush in the same rounded
box as the input.

```tsx
import { Search, X } from 'lucide-react-native';
import { TextInput } from '@nori-ui/core';

export const Example = () => (
    <>
        <TextInput leading={<Search size={16} />} placeholder="Search…" />
        <TextInput trailing={<X size={16} />} value={value} onChangeText={setValue} />
    </>
);
```

For interactive trailing content (a clear button, a visibility
toggle), wrap it in a `Pressable` — the slot accepts any React node.

## `containerClassName`

Tailwind classes for the *outer wrapper* when used without `<Field>`, targeting the
bordered field row. Use `className` on `<Field>` for grid sizing when wrapped.

```tsx
import { TextInput } from '@nori-ui/core';

export const Example = () => (
    <TextInput containerClassName="w-full md:w-80" placeholder="Subject" />
);
```
