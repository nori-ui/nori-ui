## At a glance

- Flex column with configurable `gap`, `align`, `justify`.
- Mirror of [HStack](/docs/components/hstack) for vertical layouts.
- Children stretch to the row's full width by default (cross-axis stretch). Set `align="start"` / `"center"` / `"end"` to opt out, or wrap a child in a `Box` with a fixed width.
