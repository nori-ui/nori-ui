## When to use

- **Combobox** — long lists (10+ items), typeahead filtering, async option loading.
- **[Select](/docs/components/select)** — short static lists where clicking is enough.

Combobox is a thin wrapper over Select with `searchable` defaulted to `true`. Every other prop, behavior, and type is identical — see Select's full API reference.

## Async options

Pass an `options` function that returns a promise — Select streams results into the popup as the user types. See [Select → async](/docs/components/select#async).

## API

Combobox accepts every prop of Select. The only difference is the default value of `searchable`.

See [Select's API reference](/docs/components/select#api) for the full prop list.
