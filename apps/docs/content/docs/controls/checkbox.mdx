## At a glance

- **States:** unchecked, checked, indeterminate, disabled.
- Emits semantic icon via `checkmark` key from `SemanticIconsProvider`.
- `asChild` passes control to a supplied root element.

## Indeterminate (parent / child)

A common "select all" pattern: the parent is `indeterminate` when some — but not all — children are checked. `indeterminate` is a controlled prop; toggle the parent yourself in response to the children's selection set.
