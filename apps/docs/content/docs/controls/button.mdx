## At a glance

- **Variants:** `primary`, `secondary`, `ghost`, `destructive`
- **Sizes:** `sm`, `md`, `lg`
- **States:** default, pressed, disabled, loading
- **Composition:** `asChild` renders a supplied element (e.g. `<Link>`) as the interactive root.

## asChild

`asChild` makes the supplied child the interactive root, so a Button can render as a Next.js `<Link>` (or any other element) while keeping all of its styles and accessibility behavior.

```tsx
import { Button } from '@nori-ui/core';
import Link from 'next/link';

export default function Demo() {
    return (
        <Button asChild variant="primary">
            <Link href="/somewhere">Go</Link>
        </Button>
    );
}
```
