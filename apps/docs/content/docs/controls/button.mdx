## At a glance

- **Variants:** `primary`, `secondary`, `ghost`, `destructive`
- **Sizes:** `sm`, `md`, `lg`
- **States:** default, pressed, disabled, loading
- **Composition:** `asChild` renders a supplied element (e.g. `<Link>`) as the interactive root.
