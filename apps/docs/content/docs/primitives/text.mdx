## At a glance

- **Body variants:** `body-xs`, `body-sm`, `body-md` (default), `body-lg`.
- **Heading variants:** `heading-1`, `heading-2`, `heading-3` — render with semantic heading tags + `aria-level` on web.
- Maps to RN `<Text>` on native and `<span>` / heading elements on web.
- Accepts `className` for NativeWind-driven overrides.
