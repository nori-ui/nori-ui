## At a glance

- Pulses opacity between 100% and 55% over ~900ms — calm enough to use throughout a list without flashing.
- `width` accepts a number (px) or any RN dimension string like `'40%'`. `height` is px-only.
- `radius="full"` for circular skeletons (avatars).
- `static` opts out of the animation when stacking many skeletons in a long list.
- Decorative — `aria-hidden` is set automatically. Pair with a real `aria-busy` indicator on the loading region.
