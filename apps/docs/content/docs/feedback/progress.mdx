## At a glance

- **Determinate**: pass `value` (`0..max`, default `max=100`). The fill width is `(value / max) * 100%`.
- **Indeterminate**: omit `value` to render a 30%-wide shuttle that loops across the track every ~1.5s. Use this when the operation has no measurable progress.
- **Tones**: `primary` (default, theme interactive color), `info`, `success`, `warning`, `danger` map to semantic status colors.
- **Sizes**: `sm` 4px, `md` 8px (default), `lg` 12px. The track is always pill-shaped.
- **Label**: pass `label` to render text above the bar; the percentage is auto-rendered on the right for determinate progress (set `hidePercentage` to suppress).
- Maps to `role="progressbar"` with `aria-valuemin`/`aria-valuemax`/`aria-valuenow`. Always pass `label`, `aria-label`, or `aria-labelledby` so the bar is named.
